About this app

This is 5-button calculator. The user is able to navigate and interact with the calculator using only 5 buttons on their keyboard. Users will be able to navigate up, down, left, and right through the calculator and select highlighted buttons.

  • Instructions for using the app:
    • 5-Button Controls:
      • W key: Moves highlighted button up on the calculator.
      • S key: Moves highlighted button down on the calculator.
      • A key: Moves highlighted button left on the calculator.
      • D key: Moves highlighted button right in the calculator.
      • Spacebar: Select the highlighted button.
    • 3-Button Controls:
      • S key: Moves highlighted button down on the calculator.
      • D key: Moves highlighted button right in the calculator.
      • Spacebar: Select the highlighted button.
    • 1-Button Controls:
      • Spacebar: Select the highlighted row when in row scanning mode (only far left button will be highlighted). Select the highlighted button when in column scanning mode.
      • When 1-Button Control is selected, row scanning will begin in 3 seconds.
      • You have 3 seconds on each row to decide if you want to scan the columns of that row.
      • When in column scanning, you have 3 seconds on each button to decide if you want to select it.
      • Column scanning will loop through the selected row two times before returning to row scanning.
    • NOTE: For the 3 and 1-button input methods, the SelectNext() method will not wrap around to the next row. It will instead wrap around to the beginning of the current row. This was to allow users to get buttons that they already past without having to go through all of the rows again.
    • NOTE: For the 1-button input method, the row scanning will always highlight the far left button of the row. This was to create consistency with the row scanning (user can expect the far left button to become highlighted when row scanning restarts).
